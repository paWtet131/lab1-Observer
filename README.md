# Observer TimeServer (JavaFX)

**Описание:** Учебное JavaFX-приложение, демонстрирующее паттерн **Observer (Наблюдатель)**. В роли источника данных выступает `TimeServer`: он ведёт локальный счётчик секунд и **раз в секунду** уведомляет всех подписчиков. Три наблюдателя реагируют на тики по-разному: один отображает «прошло N c», второй по расписанию показывает «клип» (в этой версии это просто **зелёный экран** на `Canvas`), третий каждые 15 секунд запускает «пульс» круга (изменение радиуса) также через `Canvas`.

**Технологический стек:** Java 17, Maven, JavaFX 17.0.6 (Controls, FXML). Desktop/Standalone GUI.

**Статус:** учебный демо-проект (Beta).

**Ссылка на продукт или демо-экземпляр:** локальный запуск через Maven (см. раздел «Установка»).

**Отличие от типовых примеров Observer:** один Subject (`TimeServer`) и сразу три разных Observer-компонента (текст/индикатор, «клип», анимация) в одном окне — без лишней логики.

## Архитектура
Ниже перечислены основные сущности (паттерн Observer + точка входа и FXML-контроллер). Пакет `com.example.lab` в `src/main/java`:
- `Subject` / `IObserver` — интерфейсы;
- `TimeServer` — реализация Subject (сервер времени, Timer/TimerTask);
- `ComponentOne`, `ComponentTwo`, `ComponentThree` — три конкретных Observer-наблюдателя;
- `HelloController` — связывает FXML и модель;
- `HelloApplication` — запуск приложения.

## Зависимости
- **JDK:** 17+ (совместимый).
- **Maven:** 3.6+ или Maven Wrapper (`mvnw.cmd`).
- **Из `pom.xml`:** `javafx-controls`, `javafx-fxml` (OpenJFX 17.0.6) — скачиваются автоматически при сборке.
- **Media/звук:** в этой реализации не требуется (ComponentTwo визуализирует «клип» зелёным экраном на `Canvas`).

## Установка
1. Откройте проект в IDE или перейдите в папку проекта.
2. Проверьте, что установлен JDK (подходит 17+).
3. Соберите проект:

```bat
mvnw.cmd clean compile
```

4. Запуск:

```bat
mvnw.cmd javafx:run
```

Если `mvnw.cmd` пишет `JAVA_HOME not found`, задайте переменную окружения `JAVA_HOME` на установленный JDK (например `C:\Program Files\Java\jdk-24`) и перезапустите терминал/IDE.

## Конфигурация
Специальная конфигурация не требуется. Интерфейс и привязка компонентов находятся в:
- `src/main/resources/com/example/lab/hello-view.fxml`
- `src/main/java/com/example/lab/HelloController.java`

## Применение
После запуска:

- кнопки **«Старт» / «Стоп» / «Сброс»** — старт/остановка/обнуление сервера времени;
- **ComponentOne** показывает «прошло N c» и рисует индикатор;
- **ComponentTwo** — задержка `N` (Spinner), **«Старт»/«Стоп»**; при достижении времени показывает «зелёный экран»;
- **ComponentThree** — каждые 15 секунд рисует «пульс» круга (изменение радиуса).
